hunana.datatypes.position module¶

class Position(position, variants, variants_flattened, variant_dict, entropy=None, variant_data=False)[source]¶

Bases: dict

Data structure for kmer positions

Describes the data structure for kmer positions within the alignment

Parameters

  • position (int) – The k-mer position number.

  • variants (Generator) – A list of type Variant containing variants seen at current position

  • variants_flattened (list) – A flattened list of all variants. Note: Only used for processing

  • variant_dict (defaultdict) – A default dictionary of lists to store the sequence idx for later getting description data

  • entropy (float) – The Shannon entropy at the current kmer position

  • variant_data (bool) – Whether to derive variant data from the sequence headers.

The attributes of this class are converted into keys when json_results=True

Attributes:

  • Position: A zero-based index for kmer position.

  • Entropy: The calculated entropy (Snannon’s Entropy) for a particular kmer position.

  • Supports: The number of valid (no gaps, no invalid characters) variants.

  • Variants: A list of variants for a particular kmer position.

  • kmer_types: Distinct variants that are different from the index variant.
