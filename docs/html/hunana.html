hunana.entropy module¶

class NormalizedEntropy(positions, max_samples=10000, iterations=10)[source]¶

Bases: object

Uses the Shannon’s Entropy formula to calculate the entropy (amount of disorder) for each k-mer position.

Parameters

  • positions (list[Position]) – A list of k-mer positions.

  • max_samples (Optional[int]) – The maximum number of samples (default: 10000)

  • iterations (Optional[int]) – The maximum number of iterations (default: 10)

run()[source]¶

This runs the entropy calculation for each of the k-mer positions generated by the Hunana module.

Returns

An iterator of Position objects with the calculated entropy as an attribute.

hunana.hunana module¶

class Hunana(seqs, kmer_len=9, header_decode=False, json_result=False, max_samples=10000, iterations=10, header_format=None, no_header_error=False, **kwargs)[source]¶

Bases: object

The Hunana algorithm returns a list of Position objects each corresponding to a kmer position.

Parameters

  • seqs (Union[str, _io.TextIOWrapper, _io.StringIO]) – A file handle, a FASTA sequence wrapped in a handle, or a filepath.

  • kmer_len (Optional[int]) – The length of the kmers to generate (default: 9).

  • header_decode (Optional[bool]) – Whether to use FASTA headers to derive kmer information (default: False).

  • json_result (Optional[bool]) – Whether the results should be returned in json format (default: False).

  • max_samples (Optional[int]) – The maximum number of samples to use when calculating entropy (default: 10000).

  • iterations (Optional[int]) – The maximum number of iterations to use when calculating entropy (default: 10).

  • header_format (Optional[str]) – The format of the header (ex: (id)|(species)|(country))

  • no_header_error (Optional[bool]) – Whether missing data in the header during header decode throws an error

(default: False)
